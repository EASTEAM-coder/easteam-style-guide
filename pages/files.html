The Six Golden Rules — apply to EVERYTHING

01

Names match the linesheet

Exactly. No abbreviations, no inventions. Wrong name = file gets sent back.

02

Underscores · no spaces · no hyphens

Only exception: compound names with + (e.g. Her_Weekend_+_Her_Outfit).

03

NEVER use Drive shortcuts

Always download + re-upload, or right-click → Make a Copy. Shortcuts break.

04

Raw assets → Production Work File 2026 first

Then copy approved edits to their permanent home. Never move.

05

Color accuracy is non-negotiable

Skirt yellow on camera = real skirt yellow. If unsure, pull the linesheet reference.

06

When in doubt, Slack Scarlett first

Faster to ask than to redo.

Folder naming pattern

[LookOrCollection]_[ContentType]_[MMDDYYYY]

File-level naming · inside each folder

Every file inside a folder inherits the folder name as its prefix. Two-digit sequence number. CamelCase inside segments.

Two-step temp rename · for batch renames

When renaming a folder of files, never rename in one pass — you'll overwrite each other. Always go through a temp prefix first.

# Step 1 — rename everything to a temp prefix
shopt -s nullglob
counter=1
for f in *.jpg; do
  mv "$f" "_tmp_$(printf '%02d' $counter).jpg"
  counter=$((counter+1))
done

# Step 2 — replace the temp prefix with the final folder name
for f in _tmp_*.jpg; do
  mv "$f" "Daydream_LifestyleEPV_05052026_${f#_tmp_}"
done
Hard rules

No spaces in filenames. No hyphens. No single-digit numbers (use 01, not 1). No Drive shortcuts. Folder name lives in every file inside it. Look names must match the linesheet exactly. If you're guessing, you're doing it wrong.
