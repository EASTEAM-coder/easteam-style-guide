Section 09 · Canonical

Email.

Warm Ivory canvas #F5F0EC. Body at 12–13px. One reason per email. The Film a Video flow is the canonical composition — every new email starts from this stack.

This page is the all-up reference. Five principles, the canonical 9-band composition, the locked universal header and footer (live previews + paste-ready spec), type and color scales sized for the inbox, the Primary Dark CTA button, the section library, layout variations, per-type recipes, do's & don'ts, and the iteration workflow. Everything you need to ship a Klaviyo email lives below.

— 01 · Five principles

If a draft contradicts any of these, rewrite or redraw before going further. The principles are non-negotiable; the section choices below are flexible.

01
Meaning in line one

The reader knows what the email is about — and what to do about it — by the end of the first sentence. No pleasantries, no preamble. The first headline is the offer or the ask.

02
One reason per email

Sell · Collect · Testimonial · Style · Restock · VIP. Pick one before writing a word. Two reasons in one email and neither lands.

03
One primary CTA

Per email — never two. Inline "shop the look" mono-pink text links inside styling cards are fine; the bottom button is the only Primary. Use Primary Dark in email (black bg, light pink text, no border).

04
Each email is its own composition

The locked header and locked footer are identical on every send. Everything in between is not. Two emails should never feel like the same template recolored — pick the section blocks and layout moves so each one reads distinct.

05
Asymmetry & breathing room

Off-balance is the brand. Center-aligned everything is off-brand. But asymmetry without padding reads as crowded — give every block enough vertical and horizontal room that the eye can rest.

— 02 · The canonical composition

The rhythm

Seven light bands sandwich one dark band (Sec-02 — Reward). The footer makes it two dark moments total — one mid-email, one closing. That's the rhythm. Don't add a second dark band in the middle, don't skip the dark Reward block. The light/dark alternation is what makes the email feel like an EASTEAM email and not a generic Klaviyo template.

— 03 · Every email exists for one reason

Pick one before you write a word. Two reasons in one email = neither lands.

01

To Sell

A new drop, restock, discount, or checkout reminder. CTA goes to a product. One CTA only.

02

To Collect

UGC video, review, photo, comment. Tell her what to send and what she gets back. "15-sec clip → $25 credit."

03

To Testimonial

A real customer story, before/after, drop debrief. Builds trust. Usually follows a Collect email by 1–2 weeks.

— 06 · Type, sized for the inbox

Three fonts, three scales. Helvetica Bold is the display font in Klaviyo email (with Arial as the Windows fallback). Staatliches is reserved for the wordmark only. Body type drops to ~13px because the inbox is dense and bigger reads as shouting. Never italic. (See typography.html for the full rules.)

Note on the Figma source: the Klaviyo Templates Figma file uses Roboto Mono Medium as a visual stand-in because Helvetica isn't available in Figma's cloud environment. The rendered email uses the CSS stack font-family: Helvetica, Arial, sans-serif; with font-weight: bold;, so the actual send uses Helvetica Bold on Mac clients and Arial Bold on Windows.

Type rules — email only

Helvetica Bold is the display font in production (Roboto Mono Medium is the Figma stand-in). Cap at 22px. Always ALL CAPS, always tracked wide. Staatliches is the wordmark only — header and footer only. Never in section headlines. Never anywhere else. Body is Jost Light 300 at 13px in the Figma file — in Klaviyo HTML the body inherits the same Helvetica / Arial stack at Regular weight so it falls back cleanly across every email client. Never italic, anywhere. If you need emphasis, use weight (Jost 400, 500 in Figma → Arial Bold in Klaviyo) or color (Deep Dusty on the last word).

Klaviyo HTML inline style for display headlines: font-family: Helvetica, Arial, sans-serif; font-weight: bold; font-size: 22px; letter-spacing: 1.76px; text-transform: uppercase; color: #0A0807;

— 08 · The CTA button

One Primary CTA per email. Always at the bottom, just above the disclaimer, centered. Primary Dark is the email default: black fill, light pink text, no border, 4px radius. The inline "shop this look" links inside styling cards are not buttons — they're underlined mono text.

Button rules — placement first

Black on black is invisible. Primary Dark goes only on the Warm Ivory canvas — never on a Page Black band. The canonical CTA section is always light, so this rarely comes up, but if you ever put a button inside a dark section (Sec-02 Reward, a dark editorial break, the footer area) the black-fill button disappears into the bg. For dark backgrounds, switch to Primary Pink: Light Pink #F9DDE1 fill with #0A0807 text, same 4px radius, no border. It reads cleanly on Page Black.

No border on either button. Fill-only — no light-pink outline. One per email. Inline "SHOP THIS LOOK →" pink mono text links inside section cards are fine and don't count as the Primary. 4px corner radius, always. On the cream canvas, default to Primary Dark. On dark sections, default to Primary Pink. Outline / Secondary / Ghost / Pill buttons from buttons.html are for the website — keep them out of email work.

— 09 · The non-negotiables

  • Body type: Jost Light 300 at 12–13px. Inbox-dense, not screen-loud.
  • One CTA per email. Use the Primary Dark button — black fill, light pink text, no border, 4px radius.
  • Meaning in the first line. Open with the reason. Not "Hi babe, hope you've been good."
  • Asymmetric layout. No center-aligned, no symmetric two-column. Off-balance is the brand.
  • Inline styles only. Gmail strips <style> tags.
  • Mobile-first single column. 600px max-width container. Stack on narrow screens.
  • Alt text on every image. Half the inboxes block images by default.
  • Occasional dark sections allowed inside an otherwise light email for editorial pull-quotes or campaign breaks.

— 13 · Section library

The blocks you stack between the locked header and locked footer. Pick a sequence that suits the email's reason — every email doesn't need every block.

— 01 · Editorial intro

The opener.

Eyebrow + 2–3 line display headline + indented body + small meta. The first thing the reader sees after the header. The meaning of the email lands here.

Canvas Warm Ivory · ~280–340h · Display Roboto Mono Medium 22px · Body Jost Light 13px indented at x=130 width=414
— 02 · Hero image band

The picture.

Full-width image, dark canvas behind. Height varies: 340 standard, 460 for editorial cover, 580–680 for product spotlight in a Back-in-Stock or VIP email.

Canvas Page Black · 340–680h · Placeholder mono Roboto Mono 10px centered "[ DESCRIPTION · RATIO · REPLACE ]"
— 03 · Numbered section

The ask / the detail / the lookbook.

The eyebrow pattern: 01 in Deep Dusty followed by — LABEL in muted, on the same line. Then a two-line display headline (often with the second line in Deep Dusty), then body.

Canvas Warm Ivory · ~260–320h · Numbered eyebrow Roboto Mono 10px +2.2px · 24px gutter between number and label
— 04 · Three-up grid (9:16)

The trio.

Three image cells in a row with one-word captions. Use for color samples, looks, lifestyle stills, sizes, or location stamps. Grid math: 488 content width = 3 × 152 + 2 × 16. Snap to that or it falls off-axis.

Canvas Warm Ivory · ~322–362h · Cells 152×200 · 16px gap · captions Jost Light 13px below
— 05 · Product catalog

The shoppable grid.

Product cards in a clean grid. 3×2 for a full drop, 2×2 for a focused sale, 1×4 for a horizontal "bestsellers" strip, 2×4 for a deep sale. Each card: image + name + color or was-to-now price. Never put pink-filled discount badges on cards.

Canvas Warm Ivory · ~410–720h · Card image 124–157 wide · name Jost Reg 13 · price Jost Light 13 (Deep Dusty for "now" price)
— 06 · Dark Sec-02 (Reward)

The dark moment.

The single dark band inside an otherwise light email. Right-aligned eyebrow + display + body, then a hairline, then a two-up stat or a piece of visual storytelling. Or carry the dark all the way through with overlaid text.

Canvas Page Black · ~380–540h · Display Roboto Mono Medium 22px in #F9EAEC and #F9DDE1 · stats Roboto Mono Light 30–48px
— 07 · CTA section

The button.

Center-aligned, the only centered section in the email. Eyebrow → 22px headline → 1-line sub → Primary Dark button. Sits second-to-last, just above the disclaimer.

Canvas Warm Ivory · 220h · Button ~300×42 · no margin between sub and button beyond 30px
— 08 · Disclaimer

The fine print.

Single-line or two-line caveat, centered, Jost Light 12px in muted gray. Specific to the email — "Made-to-order, restock not guaranteed" / "Promotion ends Sunday midnight" / "Style Notes featured looks shared with permission". Always present; never decorative.

Canvas Warm Ivory · 60–90h · Jost Light 12px · #6B6661 · centered · padding 24px top

— 14 · Layout variations

Mix and match these inside section blocks so no two emails read the same. Pick at least two distinctive moves per email; don't repeat the same move twice in one email.

— 01
Image-left, text-right

Photo bleeds off the left edge, narrow text column floats on the right. Good for a single-look feature.

— 02
Image-right, text-left

Mirror of 01. Use as the next section to zigzag the eye down the page.

— 03
Full-bleed image, text on top

Dark image fills the section. Eyebrow top-right, headline + meta bottom-left, inline CTA bottom-right. Underline aligns to the right edge of the right-aligned text.

— 04
Three-up with one-word captions

Three image tiles, identical sizing, snapped to the 488 / 3 × 152 grid. The week, the looks, the colors, the sizes.

— 05
Stamp announcement

Outlined rectangle (no fill) with "★ RESTOCKED ★" or similar inside. Use for back-in-stock or limited-run signals. Never pink-fill the stamp.

— 06
Big-number takeover

One number (the count, the discount, the countdown digit) at 48–64px Roboto Mono Light, dwarfing everything around it. Use sparingly — one per email.

— 07
Dot-grid inventory

20 small dots in a row representing 200 units of stock. Dim the dots for "sold," bright pink for "remaining." Reads as a depletion bar at a glance.

— 08
Countdown timer

Three 48px mono number blocks separated by pink colons — 47 : 23 : 14. Tiny labels under each (HRS · MIN · SEC). Right-aligned eyebrow above.

— 09
Annotated catalog

One large product image centered, four small labels around it with thin lines pointing at features (cotton-silk blend · hand-finished hem · lined throughout). Reads as a fashion-magazine spec page.

— 10
Asymmetric photo essay

One large landscape image on top, three smaller thumbnails on a clean 488 / 3 × 152 grid below. Captions are location + day, not product names.

— 11
Outlined tier card

For VIP only. Bordered rectangle (no fill), small mono labels in corners, big mono word in the middle (VIP), bottom row with member info. Like a stamped invitation, not a filled card.

— 12
Numbered perks list

Four rows, each 01 / TITLE + one-line description, with a hairline between rows. Cleaner than a card-grid for "what VIP means" or "why this drop matters."

— 15 · Per-type recipes

Each email type has one or two signature moves that make it identifiable. Build the rest from the section library and variations above.

01
Product Launch

The drop is the point. Open with a tight intro, big hero, then a real product catalog (3×2 or 2×3) followed by a colorway swatch row. Close with a dark scarcity / limited-run band.

Signature → catalog grid + 5-color swatch strip
02
Sale / Promo

The discount is the headline. Don't overstate "no code needed" — let the headline carry it. Use a strikethrough catalog (was → now in Deep Dusty), and close with a countdown timer in the dark section.

Signature → strikethrough catalog + countdown timer
03
Editorial / Lookbook

Storytelling, not selling. Open with a numbered "feeling" section at the top, follow with a pull-quote band (cream is fine, dark also fine), then an annotated-catalog or photo-essay block. End with a "read on" link, not a hard sell.

Signature → asymmetric split + pull quote + photo essay
04
Back-in-Stock

The announcement is the point. Lead with a stamp ("★ RESTOCKED ★" in a bordered box) above the headline. Big vertical product image, PDP-style info, and a visual inventory tracker (dot grid or count number) in the dark closer.

Signature → stamp opener + dot-grid inventory
05
VIP / Loyalty

Personal, not promotional. Open with a personalized Klaviyo merge tag (Dear {{ first_name }},) in the eyebrow position. Outlined tier card next, numbered perks list after, dark countdown for "doors close" timing.

Signature → outlined tier card + numbered perks list
06
Style Notes

Magazine, not catalog. One strong 1–2 line intro, hero cover image, then four styling cards in alternating layouts (image-left, full-bleed overlay, image-right, full-bleed overlay). Each card gets its own inline shop link. Close with a product grid for the literal shop.

Signature → four alternating cards + product grid

— 16 · Email do's & don'ts

Compiled from every iteration in the EASTEAM Klaviyo Templates file. Add to this list as patterns surface.

Do

  • Use Roboto Mono Medium 500 for every display headline. Cap at 22px.
  • Body in Jost Light 300 at 13px with line-height 1.7.
  • One pop color moment per surface (Deep Dusty on the last word of a title block, or Dusty Pink in a quote).
  • Stack section blocks with breathing room — 30–48px of padding between content and section edges.
  • Renumber section eyebrows to match visual order (01, 02, 03, 04 top to bottom).
  • Right-align inline CTA underlines to match right-aligned text. Left-align when text is left-aligned.
  • Use Primary Dark button (no border) — black fill, light pink text, 4px radius.
  • Style unsubscribe in disclaimer as pink underlined link with hyperlink fallback; document the Klaviyo merge tag in a designer-only mono note.
  • Mix image proportions — portrait, landscape, square, round, full-bleed — across sections to keep visual rhythm.
  • Put text on top of images when the composition allows. Don't let text always live in its own box.

Don't

  • Don't use Staatliches as a section title — it's the wordmark only.
  • Don't fill badges, stamps, pills, or sale circles in pink — outline them or skip the container entirely.
  • Don't put a border on the Primary Dark button. Black fill + light pink text, that's it.
  • Don't crowd cards — if image and text share a section, give them at least 12px between them and pad the section to 380–400h.
  • Don't repeat the same layout move twice in one email or across consecutive emails.
  • Don't run scary stats ("restock window: 4–6 weeks, if at all") — readers don't want a downer. Replace with a positive count or skip.
  • Don't leave a gap after removing a section — close it. Shift subsequent sections up and resize the frame.
  • Don't echo the footer tagline in a transition band above the footer ("MADE TO LOVE YOU" twice reads as a typo).
  • Don't propagate the retired pink palette into body sections — header and footer only.
  • Don't ship without checking the unsubscribe href is {% unsubscribe %} in the final Klaviyo HTML.

— 17 · Iteration workflow

The Klaviyo Templates Figma file is a living document. Every meaningful change becomes its own labeled iteration so older versions stay intact for reference and A/B comparison.

01
Duplicate, never overwrite

Every new change creates a new iteration frame placed below or beside the previous version. The earlier iterations stay untouched. If a "fix" doesn't need the round-trip — a typo, a single color swap — modify in place, but flag it.

02
Label every iteration

Above each new frame, put a Staatliches 20px label: 02. SALE / PROMO · ITERATION 3. Below it, a Jost Light 11px subline that summarizes the diff: 2x4 catalog · stat row removed · letter icons · button no outline.

03
Never overlap existing work

Before placing a new iteration, check what's at the target x/y. Place new frames in clearly empty regions of the canvas — typically directly below the previous version of the same template, or to the right of the v5 row. Don't drop frames on top of work-in-progress.

04
Header & footer are locked

Use the universal header and the locked footer on every new template. If the user manually changed something inside the locked header/footer of a previous iteration, copy that change forward — never silently reset it.

05
Keep names descriptive

Frames are named so the layers panel reads like an outline: Email v5 · 06 · Style Notes · iteration 5. Sections inside are Header · Editorial-Intro · Hero-img · Card-01-Morning · Sec-02-Timer · Footer. Children: EASTEAM · header-tag · pink-dot · hairline.

06
Don't fight the canvas

If the file gets cluttered, the answer is more vertical space between rows of iterations — not a smaller font on the labels. The Klaviyo Templates file is built to be browsed, not just delivered.
