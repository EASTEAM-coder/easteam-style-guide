# EASTEAM Style Guide

The Ultimate Style Guide for EASTEAM — brand, logos, color, typography, voice,
imagery, buttons, email, social, packaging, files, and more.

Live site: **https://styleguide.easteamny.com**

A static site (HTML / CSS / vanilla JS) served via GitHub Pages. The published
site is protected by a client-side access gate; enter the access code to view.

## Structure
- `index.html` — landing / section index
- `pages/` — one page per section (brand, colors, voice, email, etc.)
- `assets/` — shared `style.css`, `nav.js`, access `gate.js`, email assets
- `email-templates/`, `shopify-pages/`, `klaviyo-email*/` — reference templates

## Local preview
Open `index.html` in a browser, or run a static server from this folder.
